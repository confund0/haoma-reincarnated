# EFF Short Wordlist Attribution

The file `effshort_words.txt` in this directory is derived from the EFF
Short Wordlist #1, published by the Electronic Frontier Foundation:

  https://www.eff.org/dice

Original source file:

  https://www.eff.org/files/2016/09/08/eff_short_wordlist_1.txt

The dice-roll columns from the original source have been stripped; only
the 1296 words (one per line, sorted, lowercased) are bundled. The
selection of words is unchanged.
